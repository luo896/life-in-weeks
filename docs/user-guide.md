# 使用指南

本指南按推荐使用顺序介绍每个页面。左侧边栏（桌面）或底部标签栏（手机）切换页面。

---

## 0. 首次进入：输入生日

- **出生日期**：必填，是整张周历的基准。
- **预期寿命**：默认 90（经典人生周历取值）。也可以填一个最触动你的数字，比如 70 —— 格子越少，紧迫感越强。
- 名字可选，只用于分享卡片和报告的署名。

填完点「开始」。这些信息之后可在 **⚙️ 设置** 中随时修改。

---

## 1. 🗓️ 人生周历（主页）

**一屏看完你的一生。** 每个方块 = 一周。

| 视觉元素 | 含义 |
|---|---|
| 亮灰色方块 | 已度过的周 |
| 绿色跳动方块 | 本周（就是现在） |
| 暗色方块 | 未来的周 |
| 彩色色带 | 改善计划的里程碑（生成计划后出现） |
| 金色圆点 | 这一周写过周记 |
| 白色描边 | 当前在周记编辑器中选中的周 |

**操作**

- **悬停**任意方块：顶部显示它是第几周、对应年月、几岁、状态。
- **点击**任意方块：页面滚动到下方周记编辑器，为那一周写周记。
- **竖放 / 横放**：右上角切换。竖放 = 每行 52 周（一行一年，适合窄屏）；横放 = 每列 52 周（一列一年，适合宽屏）。不选择时按屏幕宽度自动决定，选过一次后记住你的偏好。

**下滑**可看到：

- **🎯 目标时间线**：所有目标按期望日期排列，标注「还有 N 周」。
- **📝 周记**：编辑器默认指向本周；下面按时间倒序列出全部周记，点周数可重新编辑，点 ✕ 删除。

> 周记的意义：格子会一格一格变灰，而周记让每个格子留下一句话 —— 一年后回看，你会知道那 52 个格子里发生了什么。

## 2. 📍 现状基线

**改变从诚实记录现状开始。**

- 顶部「现状一览」把你的输入实时画出来：每晚睡眠时长、体重、饮食注意项数量、24 小时轴上的睡眠时段条。
- 填写通常的入睡/起床时间（自动计算时长，支持跨午夜）、体重、运动频率、屏幕时间。
- **饮食习惯标签**：点选符合你的（常吃夜宵、含糖饮料、不吃早餐……），这是之后要"对付"的清单。
- 基线可随时更新，但建议保留第一次的诚实版本 —— 它是对比的起点。

---

## 3. 🎯 目标

四种目标类型：

| 类型 | 示例 | 说明 |
|---|---|---|
| 🌙 入睡时间 | 01:00 → 23:00 | 填过基线会自动带入现状 |
| ☀️ 起床时间 | 08:30 → 07:00 | 同上 |
| ⚖️ 体重 | 78 → 70 kg | 可自定义每周节奏，默认安全值 0.5kg/周 |
| 🎯 习惯 | 戒掉睡前刷手机 | 设定养成周期，默认 8 周 |

- 每个目标可设**期望达成日期**，它会出现在周历的目标时间线上。
- 已添加的目标显示**差距进度条**：左端现状、右端目标、圆点是你**最近一次打卡的实际值**，一眼看出完成百分比。

---

## 4. 🧭 改善计划

点「生成计划」，应用把每个目标拆成循序渐进的周任务：

- **作息**：每周提前/推后 15 分钟（温和到几乎无感，但 8 周就是 2 小时）。
- **体重**：按你设定的节奏（默认 0.5kg/周）排出每周目标体重。
- **习惯**：4 阶段养成弧线 —— 记录触发场景 → 替换行为 → 连续打卡 → 自动化。

生成后：

- 每个里程碑一张卡片，列出每周该做什么；
- 「接下来几周做什么」把所有里程碑的本周任务汇总；
- 周历上出现对应颜色的**未来色带**，你能直观看到"改变占用生命中的哪几格"。

目标变化后点「重新生成」即可。

---

## 5. ✅ 打卡

每天一分钟：

- 记录**实际**入睡/起床/体重（不是计划值 —— 诚实比好看重要）；
- 勾选完成的计划项（清单自动来自改善计划的里程碑）；
- 选心情、写一句备注。

顶部显示 🔥 连续打卡天数。补记过去某天：改日期即可；同一天重复保存会覆盖。

---

## 6. 📈 进度

- **概览**：打卡天数、连续天数、体重变化、计划坚持率。
- **体重趋势**：折线 + 目标虚线。
- **睡眠时长趋势**：目标线 8 小时。
- **入睡时间趋势**：纵轴是入睡钟点（24 = 午夜，25 = 次日 1 点），**越低越早睡**。

## 7. 🔗 分享

| 功能 | 产出 | 适合 |
|---|---|---|
| 分享卡片 | 1080×1350 PNG：周历缩略图 + 关键成果 | 发朋友圈/社交媒体 |
| 过程报告 | 单文件 HTML：基线→目标→计划→打卡→周记 全过程 | 存档、发给朋友、⌘P 打印成 PDF |
| 导出备份 | 完整 JSON | 换设备、防丢失 |
| 导入备份 | 从 JSON 恢复（**覆盖**当前数据） | 迁移 |

> ⚠️ 数据只在本机浏览器。**定期导出 JSON 备份。**

## 常见问题

**改了出生日期，周记会怎样？**
周记按「第 N 周」存储，N 从出生日期数起。改生日后周序号不变，但对应的日历日期会平移。

**预期寿命填小了，已经超龄会怎样？**
统计显示已走过 100%、剩余 0 周；「本周」标记落在最后一格，一切功能正常。

**想删掉某周周记？**
周记列表里点 ✕。（编辑器清空文字保存不会删除，防误删。）

**换电脑怎么迁移？**
旧机器：分享 → 导出备份（JSON）。新机器：完成首次生日输入后，分享 → 导入备份。
