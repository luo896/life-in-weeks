# 更新日志

## v1.3.0 — 2026-07-04

### 新增
- **中英双语（i18n）**：全部界面、改善计划生成文案、分享卡片、过程报告、导出文件名均支持中文/英文。默认跟随系统语言，可在「设置 → 语言」切换（自动/中文/English）；引导页右上角也可直接切换。
- 零依赖实现：`src/lib/i18n.jsx` 词典 + 轻量 `t()` 插值，`<html lang>` 与标签页标题随语言同步。

### 说明
- 改善计划中的文案在**生成时**按当前语言写入并保存；切换语言后点「重新生成」即可刷新计划语言。历史打卡的勾选项名称与已选饮食标签保留原语言。

### 部署
- 新增 GitHub Pages 自动部署（`.github/workflows/deploy.yml`），推送 `main` 自动发布到 <https://luo896.github.io/life-in-weeks/>。

## v1.2.0 — 2026-06-29

### 修复
- **超龄边界**：实际年龄超过设定预期寿命时，「本周」标记现在正确落在最后一格（此前不显示，且周记编辑器指向越界周）。
- **周历方向自动化**：未手动选择横竖方向时按屏幕宽度自动选择（≤860px 竖放，否则横放），并随窗口缩放实时切换；此前移动端默认横放导致格子过小。
- **过程报告补全**：HTML 过程报告新增「⑤ 周记」章节；分享页数据摘要加入周记计数。

### 文档
- 新增 docs/：使用指南、架构设计、数据模型、开发文档；重写 README。

## v1.1.0 — 2026-06-29

### 新增
- **周历一屏自适应**：格子尺寸按视口实时计算，整个人生周历完整落在首屏。
- **横放 / 竖放切换**：右上角切换周历方向，选择持久化。
- **周记**：点周历任意格子为那一周写周记；金色圆点标记有周记的周；周历下方新增周记编辑器与历史列表。
- **目标时间线**：周历下方按期望日期排列所有目标，显示「还有 N 周」。
- **现状可视化**：基线页新增「现状一览」（睡眠时段 24 小时条 + 关键数字）。
- **目标进度可视化**：每个目标显示「现状 → 最近打卡 → 目标」差距进度条与完成百分比。

### 修复
- 周历「有周记」标记与「选中」标记视觉混淆：周记改为金色圆点（固定），选中为白色描边（跟随点击），二者独立。
- 提升已度过/未来格子的对比度，暗色背景下整表清晰可辨。

## v1.0.0 — 2026-06-28

首个版本：

- 🗓️ 人生周历（~90×52 周格），已度过/本周/未来三态，计划色带叠加
- 📍 现状基线：作息（自动算时长）、体重、饮食标签、运动、屏幕时间
- 🎯 目标：入睡/起床/体重/习惯四类，基线自动带入
- 🧭 改善计划：自动拆解为逐周小步（作息 15 分钟/周、体重 0.5kg/周、习惯 8 周弧线）
- ✅ 每日打卡：作息/体重/计划完成项/心情/备注，连续天数统计
- 📈 进度：体重/睡眠时长/入睡时间趋势图（含目标线）、坚持率
- 🔗 分享：PNG 分享卡片、HTML 过程报告（可打印 PDF）、JSON 备份/导入
- 技术：React 18 + Vite 6，零运行时依赖，数据全部本地 localStorage
