## v1.6.0 — 2026-07-06「循证与锚点」

## v1.5.0 — 2026-07-04

### 新增
- **GitHub 账号登录 + 多设备同步**：设置或引导页登录后，数据自动同步到云端（Cloudflare Worker + D1），换设备登录即恢复；不登录则一切如旧、数据只存本机。
- 同步为「本地优先」：启动 / 窗口聚焦 / 网络恢复时拉取，修改后 1.5 秒防抖推送；两台设备并发修改时按条目合并（周记 / 打卡 / 目标逐条取较新者），不会互相覆盖。
- 新增后端 `server/`（零依赖 Cloudflare Worker）：GitHub OAuth 授权码流程、HMAC 签名会话 token（30 天）、带乐观锁版本号的快照读写（冲突返回 409 由客户端合并重推）。
- 顶栏新增同步状态指示（☁️ 已同步 / 🔄 同步中 / 📴 离线 / ⚠️ 出错），设置内可查看账户、立即同步、退出登录；界面文案三语齐备。

### 说明
- 隐私口径更新：数据**默认只存本机浏览器**；只有登录后才会同步到应用的云端数据库。退出登录不清除本机数据。
- 横竖排方向与语言属于设备偏好，不参与同步（每台设备各自记忆）。
